# 更新日志

本文档记录项目的所有重要更改。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [未发布]

### 新增
- 初始版本发布
- 书名输入功能（10个排名位置）
- 豆瓣API集成搜索书籍
- **图片搜索功能** - 支持百度图片搜索作为豆瓣API的备选方案
- **双搜索模式** - 用户可在豆瓣搜索和图片搜索之间切换
- 书籍选择和预览
- 拖拽排序功能
- 自定义封面上传
- 样式自定义（背景色、文字色、标题）
- 三种预设模板（经典、现代、简约）
- Canvas图片生成引擎
- PNG和JPG格式导出
- LocalStorage数据持久化
- 响应式设计（桌面端和移动端）
- 键盘快捷键支持
- 加载动画和反馈提示
- 错误边界和错误处理
- 全面的单元测试
- 属性测试（Property-Based Testing）
- 端到端集成测试

### 改进
- 搜索弹窗支持标签页切换
- 图片搜索结果以网格形式展示
- 自动在搜索查询中添加"封面"关键词
- 降级策略：搜索失败时使用模拟数据

### 技术栈
- React 19.2
- TypeScript 5.9
- Vite 7.2
- Tailwind CSS 4
- Vitest 测试框架
- fast-check 属性测试
- @dnd-kit 拖拽功能

## 版本说明

### 版本号格式：主版本号.次版本号.修订号

- **主版本号**：不兼容的API修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 更新类型

- **新增**：新功能
- **变更**：现有功能的变更
- **弃用**：即将移除的功能
- **移除**：已移除的功能
- **修复**：Bug修复
- **安全**：安全相关的修复
